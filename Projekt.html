Źródło danych¶

https://comtrade.un.org/data/

Analiz przeprowadzono na podstawie danych dotyczcych eksportu midzy Polsk i pastwami z ni graniczcymi, a pozostaymi krajami. Dane zostay przygotowane przez United Nations (ONZ) i dotycz roku 2018 zawierajc (po wstpnym przetworzeniu przy uyciu programu CSVed) kolejno: pastwo eksportujce, pastwo importujce, sumaryczny koszt transakcji podany w US$.

g=nx.from_pandas_edgelist(data,
                          source='Reporter',
                          target='Partner',
                          edge_attr='Trade Value (US$)',
                          create_using=nx.Graph())

countries = dict(g.degree(weight='Trade Value (US$)')).keys()
countries_ordered = sorted(countries)

datas = []

for key in countries_ordered:
    data = {}
    data['Kraj'] = key
    data['Liczba partnerw'] = len(g.edges(key))
    edges = [tuple[1] for tuple in list(g.edges(key))]
    data['Lista partnerw'] = '\n'.join(sorted(edges))
